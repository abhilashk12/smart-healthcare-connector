# Smart Healthcare Connector

An AI-powered healthcare authorization platform enabling seamless communication between Providers and Payers using FHIR standards.

## Tech Stack

Backend
- Java 21
- Spring Boot
- Spring Security
- JPA
- MySQL

Frontend
- Angular 18
- Angular Material
- RxJS

AI
- OpenAI/Gemini/Ollama

Documentation
- Swagger

Authentication
- JWT
